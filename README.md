# muzzy

## prolog

muzzy (combination of "meme" and "buzzy") - your favorite meme sharing web app.

## how to contribute

Start up redis

```
docker-compose up
```

Web app + API

```
npm start
```

## run tests

React components unit tests:

```
npm test
```

Cypress powered UI interactivity (mainly dropzone) testing:

```
npm run e2e
```
